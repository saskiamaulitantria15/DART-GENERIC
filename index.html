📖 Rangkuman Materi Generic (Slide 1–16)

  • Generic : fitur untuk membuat kode bekerja dengan berbagai tipe data namun tetap aman terhadap tipe.
  • Manfaat : type safety (compile-time), code reusability, tanpa casting manual.
  • Sintaks : class Box<T> { T data; } , fungsi T myFunc<T>(T value).
  • Multiple parameter : class Pair<K, V>.
  • Batasan tipe : extends (misal T extends num).
  • Type inference : Dart bisa menebak tipe generic dari konteks.
  • Perbandingan : Generic lebih aman dibanding dynamic atau Object.
  • Contoh nyata : List<E>, Map<K,V>, Cache<T>.

Apa itu Generic?

Generic adalah fitur yang memungkinkan kelas, fungsi, atau interface dapat bekerja dengan berbagai tipe data tanpa mengorbankan keamanan tipe. Dart mendukung generic dengan type parameter seperti <T>.

Manfaat:

  • Type safety – error terdeteksi saat kompilasi
  • Code reusability – satu kelas untuk banyak tipe
  • Compile-time checking – tidak perlu casting manual

Masalah Tanpa Generic (dynamic)

Tanpa generic, Box menggunakan dynamic sehingga fleksibel tetapi tidak aman karena berpotensi menimbulkan runtime error.

Sintaks Generic (Class & Fungsi)

Generic pada class dan fungsi memungkinkan kode digunakan kembali untuk berbagai

tipe data seperti int, String, atau double tanpa membuat kode baru, sehingga program menjadi

lebih fleksibel, reusable, dan aman dari kesalahan tipe data.

// Nama: Saskia Mauli Tantria
// NIM: 251410034
// Kelas: SI2A

class MyClass<T> {
  T data;

  MyClass(this.data);

  T getData() {
    return data;
  }
}

T myFunc<T>(T value) {
  return value;
}

void main() {
  // Class Generic
  var myIntClass = MyClass<int>(10);
  var myStringClass = MyClass<String>('test');

  print(myIntClass.getData());    // Output: 10
  print(myStringClass.getData()); // Output: test

  // Fungsi Generic
  var intValue = myFunc<int>(100);
  var stringValue = myFunc<String>('hello');

  print(intValue);    // Output: 100
  print(stringValue); // Output: hello
}

Contoh kelas Box <T>

Class Generic Box <T> memungkinkan penyimpanan berbagai tipe data seperti int, String, dan double dalam satu

class yang sama, sehingga kode menjadi lebih fleksibel, reusable, dan aman dari kesalahan tipe data.”

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A 

class Box<T> {
  T? _value; // Tipe data T, nullable

  Box(this._value); // Constructor

  T? get value => _value; // Getter

  void setValue(T value) {
    _value = value;
  }
}

void main() {
  // Menggunakan Box dengan tipe data int
  var intBox = Box<int>(10);
  print('Int Box Value: ${intBox.value}');
  // Output: Int Box Value: 10

  intBox.setValue(20);
  print('Int Box Value After Set: ${intBox.value}');
  // Output: Int Box Value After Set: 20

  // Menggunakan Box dengan tipe data String
  var stringBox = Box<String>('Hello, Generic!');
  print('String Box Value: ${stringBox.value}');
  // Output: String Box Value: Hello, Generic!

  stringBox.setValue('Hello World');
  print('String Box Value After Set: ${stringBox.value}');
  // Output: String Box Value After Set: Hello World

  // Menggunakan Box dengan tipe data double
  var doubleBox = Box<double>(3.14);
  print('Double Box Value: ${doubleBox.value}');
  // Output: Double Box Value: 3.14
}

Multiple Type Parameter: Pair<K,V>

Generic dengan multiple type parameter Pair<K,V> memungkinkan satu class menyimpan dua data dengan tipe berbeda,

seperti String dan int atau int dan double, sehingga kode menjadi lebih fleksibel, reusable, dan tetap type-safe.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

class Pair<K, V> {
  K key;
  V value;

  Pair(this.key, this.value);

  void printPair() {
    print('Key: $key, Value: $value');
  }
}

void main() {
  // Pair dengan tipe data String dan int
  var pair1 = Pair<String, int>('age', 30);
  pair1.printPair(); // Output: Key: age, Value: 30

  // Pair dengan tipe data String dan String
  var pair2 = Pair<String, String>('name', 'Alice');
  pair2.printPair(); // Output: Key: name, Value: Alice

  // Pair dengan tipe data int dan double
  var pair3 = Pair<int, double>(1, 2.71);
  pair3.printPair(); // Output: Key: 1, Value: 2.71
}

Menggabungkan dua nilai dengan tipe data yang berbeda.

generic pada fungsi combine<T,U> memungkinkan penggabungan dua nilai dengan tipe data berbeda, seperti String, int, atau double,

sehingga fungsi menjadi lebih fleksibel, reusable, dan tetap type-safe.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

String combine<T, U>(T value1, U value2) {
  return '$value1, $value2';
}

void main() {

  print(result1); // Output: Number is , 10

  var result2 = combine<int, double>(10, 20.5);
  print(result2); // Output: 10, 20.5
}

Type Inference (Inferensi Tipe)

Type Inference membuat Dart bisa menebak tipe data secara otomatis, sehingga penulisan kode menjadi lebih sederhana dan tetap aman
//Nama: Saskia MauliTantria
//NIM: 251410034
//Kelas: SI2A

T myFunc<T>(T value) {
  return value;
}

void main() {
  var intValue = myFunc(10); // Dart infer T = int
  print(intValue.runtimeType); // Output: int

  var stringValue = myFunc("test"); // Dart infer T = String
  print(stringValue.runtimeType); // Output: String
}

Pembatasan tipe dan manfaatnya

Lebih aman (type-safe): mengurangi error saat program jalan

Kode lebih jelas: tipe data terlihat langsung

Bisa pakai method khusus: sesuai tipe yang dibatasi

Mencegah tipe salah: tidak bisa asal input data

Desain lebih rapi: lebih terstruktur saat buat generic

Pembatasan Tipe pada Interface dan Class

Generic extends membatasi tipe data yang digunakan, sehingga fungsi hanya menerima data yang sesuai dan lebih aman dari kesalahan tipe.

Program Sukses

Program berhasil karena data yang dikirim (List dan Set) merupakan turunan dari Iterable, sehingga memenuhi batasan T extends Iterable<dynamic> dan dapat diproses dengan perulangan.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

void printIterable<T extends Iterable<dynamic>>(T iterable) {
  for (var item in iterable) {
    print(item);
  }
}

void main() {
  printIterable<List<int>>([1, 2, 3]);
  printIterable<Set<String>>({'apple', 'banana'});
}

Pembatasan Tipe dengan Class Abstract

Pembatasan tipe dengan class abstract digunakan untuk membatasi Generic agar hanya menerima class yang merupakan turunan dari abstract class tersebut, sehingga penggunaan tipe data menjadi lebih terarah dan aman.

Program Sukses

Program sukses karena Dog dan Cat merupakan turunan Animal, sehingga memenuhi batasan T extends Animal.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

abstract class Animal {
  void makeSound();
}

class Dog implements Animal {
  @override
  void makeSound() {
    print('Woof!');
  }
}

class Cat implements Animal {
  @override
  void makeSound() {
    print('Meow!');
  }
}

class AnimalCage<T extends Animal> {
  T animal;

  AnimalCage(this.animal);

  void sound() {
    animal.makeSound();
  }
}

void main() {
  var dogCage = AnimalCage<Dog>(Dog());
  dogCage.sound();

  var catCage = AnimalCage<Cat>(Cat());
  catCage.sound();
}

Praktik 1 - Generic Class dan fungsi

1. Buat class Repository <T> dengan:

Method: save(T item) , T getById(int id) , List <T> getAll()

2. Buat fungsi generic swap<T>(List<T> list, int i, int j)

3. Buat class Triple<A, B, C> dengan 3 field

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A 

class Repository<T> {
  final Map<int, T> _data = {};

  void save(int id, T item) {
    _data[id] = item;
  }

  T? getById(int id) {
    return _data[id];
  }

  List<T> getAll() {
    return _data.values.toList();
  }
}

void swap<T>(List<T> list, int i, int j) {
  T temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

class Triple<A, B, C> {
  A first;
  B second;
  C third;

  Triple(this.first, this.second, this.third);
}

void main() {
  // Repository
  var repo = Repository<String>();
  repo.save(1, "Halo");
  repo.save(2, "Dunia");

  print(repo.getById(1));
  print(repo.getAll());

  // swap
  var list = [1, 2, 3];
  swap(list, 0, 2);
  print(list);

  // Triple
  var triple = Triple<String, int, bool>("Umur", 20, true);
  print(triple.first);
  print(triple.second);
  print(triple.third);
}

Generic vs dynamic vs Object

Type Safety: Generic aman (dicek saat compile), dynamic tidak aman (runtime), Object cukup aman tapi terbatas.

Flexibility: dynamic paling fleksibel, Generic mengikuti tipe (terbatas), Object juga terbatas.

Performance: Generic dan Object optimal, dynamic lebih lambat karena ada overhead.

Casting: Generic & dynamic tidak perlu casting, Object perlu casting.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A
      
 class Box<T>{
  T data;

  Box(this.data);
}

void main() {
  // Contoh perbedaan
  Box<int> box1 = Box<int>(42); // Generic: aman
  dynamic box2 = 42;// Dynamic: tidak aman
  Object box3 = 42;// Object: perlu casting

  print(box1.data);
  print(box2);
  print(box3);
}

Studi kasus - Cache Generic

Generic pada class Cache<T> digunakan untuk menyimpan berbagai tipe data, seperti String dan object Product, dalam satu class yang sama, sehingga kode lebih fleksibel, reusable, dan aman dari kesalahan tipe data

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

class Cache<T> {
  final Map<String, T> _storage = {};

  void set(String key, T value) {
    _storage[key] = value;
  }

  T? get(String key) {
    return _storage[key];
  }

  void remove(String key) {
    _storage.remove(key);
  }
}

class Product {
  String name;
  int price;

  Product(this.name, this.price);
}

void main() {
  var userCache = Cache<String>();
  userCache.set('name', 'Alice');

  var productCache = Cache<Product>();
  productCache.set('p1', Product('Laptop', 1000));

  print(userCache.get('name')); // Alice
  print(productCache.get('p1')?.name); // Laptop
}

//Nama: Saskia Mauli Tantria

class Stack<T> {
  final List<T> _stack = [];

  void push(T value) {
    _stack.add(value);
  }

  T? pop() {
    return _stack.isNotEmpty ? _stack.removeLast() : null;
  }

  T? peek() {
    return _stack.isNotEmpty ? _stack.last : null;
  }
}

List<T> mergeLists<T>(List<T> a, List<T> b) {
  return List<T>.from(a)..addAll(b);
}

class Response<T> {
  T? data;
  String? error;

  Response({this.data, this.error});
}

// Testing
void main() {
  // Stack
  var stack = Stack<int>();
  stack.push(5);
  stack.push(10);
  print(stack.peek()); // 10
  print(stack.pop());  // 10

  // Merge Lists
  var hasil = mergeLists([1, 2], [3, 4]);
  print(hasil); // [1,2,3,4]

  // Response
  var res1 = Response<String>(data: "OK");
  var res2 = Response<String>(error: "Gagal");

  print(res1.data);
  print(res2.error);
}
